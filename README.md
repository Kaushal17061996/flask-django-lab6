# Flask and Django Web Frameworks Lab

## Overview
This project demonstrates the implementation of two Python web frameworks:
Flask and Django. The objective was to understand routing, templating,
and basic web application development.

---

## Part 1: Flask Application

The Flask application includes:

- A home route (/) that displays "Welcome to Flask!"
- A dynamic route (/hello/<name>) that displays a personalized greeting
- A form page where users can enter their name
- A template system using Jinja2

### How to Run Flask App

1. Navigate to the flask project folder
2. Activate the virtual environment
3. Run the application
